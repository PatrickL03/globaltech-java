# Sistema Empresarial GlobalTECH

Proyecto académico desarrollado en Java enfocado en la gestión de empleados por departamentos.

## Funciones
- Gestión de empleados
- Clasificación por áreas
- Organización modular
- Aplicación de Programación Orientada a Objetos

## Tecnologías
- Java
- NetBeans
- Programación Orientada a Objetos
